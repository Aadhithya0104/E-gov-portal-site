# Tamil Nadu E-Governance Portal 🇮🇳

A full-featured Angular 19 E-Government portal for Tamil Nadu with citizen services, grievance management, and admin dashboard.

## Features
- 3-type login: Citizen, Admin, New User (OTP)
- 12+ Government services (Aadhaar, Ration Card, Voter ID, Patta & Chitta, etc.)
- PDF document upload
- Application status tracking
- Admin dashboard with issue management
- Real-time status updates visible to citizens
